# FAQ Chatbot using NLP

## Project Overview

The FAQ Chatbot is a Natural Language Processing (NLP) project developed using Python. It answers user questions by comparing them with a predefined FAQ dataset and returning the most relevant answer using text similarity techniques.

## Features

* FAQ dataset with questions and answers
* Text preprocessing using NLTK
* TF-IDF Vectorization
* Cosine Similarity for question matching
* Interactive chatbot interface using Gradio
* Simple and user-friendly design

## Technologies Used

* Python
* Google Colab
* Pandas
* NLTK
* Scikit-learn
* Gradio

## Project Workflow

1. Create an FAQ dataset.
2. Preprocess the text using NLP techniques.
3. Convert the processed text into TF-IDF vectors.
4. Compare the user's question with FAQ questions using Cosine Similarity.
5. Display the most relevant answer through the chatbot interface.
